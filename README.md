## Getting Started

To run the IOS CALCULATOR on your local machine, follow these steps:

1. Clone this repository:

```bash
  git clone https://github.com/enescopler/ios-calculator.git
```

2. Open the project in VsCode:

```bash
  cd ios-calculator
  code . 
```
